# SkwnFrontendDev Project

Repository ini merupakan hasil pengerjaan test Frontend Developer.

## How to start project

1. Pastikan Anda telah menginstal [Node.js](https://nodejs.org/) di sistem Anda.
2. Clone repository ini.
3. Buka terminal pada folder proyek, lalu jalankan perintah berikut untuk menginstal dependensi:
   ```bash
   npm install
   ```
4. Setelah instalasi selesai, jalankan server environment lokal menggunakan perintah:
   ```bash
   npm run dev
   ```
5. Buka link localhost yang muncul di terminal (biasanya `http://localhost:5173/`) di browser Anda.

## Teknologi yang digunakan

- HTML / CSS (Responsive & Animations diutamakan)
- Vanilla JavaScript
- [JQuery](https://jquery.com/) untuk manipulasi DOM & Ajax API Request.
- [Vite.js](https://vitejs.dev/) untuk local dev server environment.
- Mock API: [JSONPlaceholder](https://jsonplaceholder.typicode.com/)
